# 🚀 TagCluster

TagCluster é um SaaS de criação, gestão e impressão de etiquetas com layouts personalizáveis e controle de rastreabilidade.

## 🎯 Objetivo

Permitir que empresas e usuários:

- Criem etiquetas personalizadas
- Gerenciem modelos
- Imprimam ou exportem em PDF
- Rastreiem impressões
- Organizem por pastas
- Trabalhem com dados em lote
